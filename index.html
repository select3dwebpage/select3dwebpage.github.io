Introduction


The VR pointer adapter enables developers to easily add gaze pointers to a Unity project. The same scripts will work with all available platforms.

This adapter adds the ability to interact with the environment either by using eye tracking or head tracking. The pointer allows you to make GameObjects gaze-aware where event listeners are called whenever the player looks at them. This can be used to create an intuitive interaction system or to spread the functionality to a new source of input so that your controllers can be used to do something else.

This adapter works with Steam VR camera.


How to Use the Adapter


  • Add a platform variable to the script to select the type of platform you want to run this script on.
  • Select Generic for a normal HTC Vive to add head tracking based selector and SMI to run the script on a SMI HTC Vive.
  • Add the following code snippet to the start method to set-up the adapter:

gameObject.AddComponent();
adapter = gameObject.GetComponent();
adapter.setTarget(this.gameObject);
adapter.setPlatform(platform); // platform is a public variable.

Add GazeInterface to the script and import all the required methods from the interface.
Implement methods: Implement the OnPointerEnter, OnPointerExit and OnPointerStay methods
